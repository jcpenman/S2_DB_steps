In order to run the scripts in this folder you will require SYSTEM, EDM and SGAS login details for the STEPS database.

Please observe the run order below for executing the scripts in this folder.

1. As the SYSTEM user, run BT_BUILD_ILA500_SCHEMA.sql
(This will run ILA_500_SCHEMA_BT.sql)

2. As the ILA500 user run BT_BUILD_ILA500_TABLES.sql
(This will run LEARNER.sql and PROVIDER.sql and PROVIDER_INSERT.sql)

3. As the EDM user run BT_BUILD_ILA500_EDM_DATA.sql
(This will run EDM_BATCH_WF_INSERT.sql)

4. AS the SGAS user run BT_BUILD_ILA500_SGAS_DATA.sql
(This will run CONFIG_EDM_INSERT.sql and CONFIG_DATA_INSERT.sql)
